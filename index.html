Sobre mim

Desde pequeno, a arte sempre fez parte da minha vida. Passava horas desenhando, criando e dando vida às minhas ideias no papel. Em 2021, entrei para o curso técnico de Informática para Internet e, ali, comecei a explorar um novo tipo de criação: o mundo da tecnologia.

Em 2024, decidi seguir meu coração e me dedicar ao desenvolvimento Front-End. Descobri que programar é, de certa forma, como desenhar com lógica. É transformar ideias em algo real, visual, interativo. Eu amo começar do zero, ver cada detalhe ganhando forma e, no fim, sentir aquele orgulho de ter construído algo com as próprias mãos e linhas de código.

Projeto Milk Shake

Desenvolvido a partir de uma vídeo aula, este projeto foi adaptado para criar um carrossel interativo exibindo opções de milkshake de morango, chocolate e paçoca. Utilizando HTML, CSS e JavaScript, o objetivo foi praticar conceitos de estrutura, estilização e interatividade no front-end.

Projeto Clínica Odontológica

Landing page desenvolvida com HTML e Bootstrap, focada em clínicas dentárias. A página apresenta de forma organizada os serviços oferecidos como clareamento, ortodontia e implantes, além de depoimentos reais de pacientes, criando uma experiência visual atraente e acessível. O uso do Bootstrap garante responsividade e compatibilidade com diferentes dispositivos, aprimorando a navegação e o visual geral

Projeto Studio Ghibli

Este projeto é uma aplicação web interativa construída com HTML, CSS e JavaScript que consome a API do Studio Ghibli para apresentar conteúdos sobre os filmes, personagens, fundadores e prêmios relacionados ao estúdio, de forma organizada e visualmente atrativa. Com foco na manipulação dinâmica de dados por meio de chamadas à API, o projeto demonstra habilidades em integração com APIs RESTful e na geração de conteúdo em tempo real no front-end
